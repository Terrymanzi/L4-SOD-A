# World Mission Cultural Day

## Event Registration System Project

### Project Overview

Your task is to design and build a **fully functional event registration system** for the school’s **World Mission Cultural Day** using:

* HTML
* CSS
* JavaScript

This project will simulate a real-world scenario where users (students, teachers, and guests) register for an event online.

---

## Learning Objectives

By completing this project, you should be able to:

* Structure a webpage using HTML forms
* Style user interfaces using CSS
* Add interactivity and validation using JavaScript
* Handle user input dynamically
* Store and display data (using Local Storage or arrays)
* Think like a real-world web fullstack developer

---

## Project Requirements

### 1. Registration Form

Create a form that collects the following information:

* Full Name
* Email Address
* Phone Number
* Gender (Radio buttons)
* Country / Culture Represented
* Event Participation Type (Dropdown)

  * Performer
  * Volunteer
  * Attendee
* Special Requests (Textarea)
* Terms & Conditions (Checkbox)

---

### 2. Form Validation (JavaScript Required)

You must validate:

* Required fields are not empty
* Email format is correct
* Phone number is valid (numbers only, reasonable length)
* Terms checkbox is checked before submission

Display **clear error messages** when validation fails.

---

### 3. Data Handling

When a user submits the form:

* Prevent page reload
* Save the data using:

  * JavaScript arrays **or**
  * Local Storage (recommended)
* Display a success message

---

### 4. Registration List Display

Create a section/page that shows:

* List of registered participants
* Each entry should display:

  * Name
  * Participation Type
  * Country
* Add a **Delete button** for each entry

---

### 5. UI/UX Design (CSS)

Inspiration: [click here](https://w3layouts.com/wp-content/uploads/2017/07/event_registration_form_Free29-06-2017_1219919468.jpg)

Your design should:

* Be visually appealing and culturally themed
* Use colors inspired by global cultures
* Be responsive (works on mobile and desktop)
* Have proper spacing, alignment, and readable fonts

---

### 6. Bonus Features (Optional but rewarded)

Try adding:

* Search functionality
* Filter by participation type
* Edit/update registration
* Animations or transitions
* Export registrations (advanced)
